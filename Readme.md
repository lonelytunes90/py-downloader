# 🚀 Universal Media Downloader

A powerful, cross-platform Python script to download Video and Audio from YouTube, Facebook, and 1000+ other websites.

## ✨ Features
- **Universal:** Works on Termux (Android), Windows, macOS, and Linux.
- **Smart Sorting:** Automatically organizes downloads into `muzika`, `dramas`, and `web` folders.
- **High Quality:** Downloads full audio (320kbps MP3) and best available video (MP4).
- **Playlist Support:** Downloads entire playlists with numbered tracks.
- **Custom Paths:** Choose exactly where your files go.

## 🛠️ Requirements
- [Python 3.x](https://www.python.org/)
- [FFmpeg](https://ffmpeg.org/) (Required for MP3 conversion)

## 🚀 Installation
1. Clone this repo:
   ```bash
   git clone https://github.com/lonelytunes90/py-downloader.git
   ```
   cd py-downloder


## 2. Install dependencies:
